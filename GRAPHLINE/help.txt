Отображение функций
1) для отображения графика функции необходимо нажать на кнопку ОК рядом с полем ввода.
2) для удаления функции с графика необходимо ввести её и нажать кнопку CANCLE
3) максимальное количесво отображаемых функций - 5.

сохранение функций
1) все вводимые функции автоматически сохраняются в базу данных при нажатии кнопки OK
2) сохранить любую функцию в базу данных, нажав кнопку SAVE

история вводимых функций
1) для того, чтобы посмотреть историю вводимых функций необходимо нажать кнопку
HISTORY
2) для просмотра сохраненных пользователем функций достаточно нажать кнопку saved

параметры
1) для ввода параметра необходимо нажать кнопку ADD PARAMETER
   далее необходимо ввести три значения: начало, конец и шаг
   Внимание! Необходимо ввести все три значения!
   по умолчанию начало = 1, конец = 10, шаг = 1
   
   warning: начало должно быть меньше чем конец
   для отображения графика функции с параметром нажмите OK и
   передвигайте ползунок слайдера.
